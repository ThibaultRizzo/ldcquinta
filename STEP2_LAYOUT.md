## Scrolling Behavior

### Desktop/Tablet
- **Left Sidebar**: Scrolls only vertically (when services exceed height)
- **Form Area**: Scrolls independently (right section only)
- **Footer**: Always visible, doesn't scroll

### Mobile
- **Services**: Horizontal scroll (if needed)
- **Form**: Vertical scroll with full width
- **Footer**: Fixed at bottom, doesn't scroll

## Accessibility Features

### Visual Hierarchy
1. **Title**: 18px, bold, gradient color
2. **Labels**: 12px, bold, uppercase, required indicator (*)
3. **Inputs**: 13px, standard weight
4. **Help text**: 11px, lighter gray

### Focus States
- Border color: `#667eea`
- Box shadow: `0 0 0 3px rgba(102, 126, 234, 0.1)`
- Visible on Tab/keyboard navigation

### Keyboard Navigation
- Tab: Moves through form fields
- Enter: Submits form
- Escape: Could close language menu
